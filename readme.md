## What I’ve Worked On

### Smart IoT Door System  
A multi-layer authentication system using QR codes, fingerprint, face recognition, RFID, and sensors to detect things like tailgating and motion. This was more about combining everything into one working system than just making individual features.

### Smart Fire Alarm System  
Built around Raspberry Pi with sensors and real-time alerts through a Telegram bot. Focus was on reliability and fast response rather than complexity.

### Data Analytics Project (Life Expectancy)  
Worked with R Studio and Spotfire to clean, process, and analyse global health data. Main focus was on turning messy datasets into something meaningful and presentable.

### Android Projects  
A few smaller apps built in Android Studio — mostly focused on learning how to structure apps, handle JSON data, and design usable interfaces.
